# 20260615_028 — 득점 순위 → 선수 순위 (골·어시스트·공격P, 정렬)

## 변경
- 기존 "득점 순위"를 **선수 순위**로 개편. URL(`matches:scorers`, `/matches/scorers/`)은 유지.
- `apps/matches/views.py` `scorers` — 우리 팀 GOAL+ASSIST 이벤트를 선수별 집계:
  `goals`, `assists`, `points`(=골+어시) 컬럼. 기본 정렬 공격P 내림차순. 팀·대회·시즌 필터 유지.
- `templates/matches/scorers.html` — 순위·선수·골·어시스트·공격P 테이블.
  - **헤더 클릭 정렬**(골/어시스트/공격P): 바닐라 JS로 행 재정렬(asc/desc 토글, ▲▼ 표시).
  - 순위는 CSS 카운터라 정렬해도 1,2,3… 자동 갱신. 숫자는 Bebas(.num).
- `base.html` 네비 라벨 "득점 순위" → "선수 순위".

## 비고
- 정렬은 클라이언트(JS)에서 즉시 처리(페이지 리로드 없음). 필터는 기존대로 서버 처리.
- 별도 JS 빌드 없음(인라인 스크립트).

## 테스트
- 페이지 200, 골·어시·공격P 집계 DB와 일치(엄길헌 5골2어시=7P 등), 18명, 기본 공격P desc.
