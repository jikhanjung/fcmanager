# 20260615_021 — navbar 이미지 버전 표시

## 변경
- navbar 브랜드 "FC Sky" 옆에 배포 이미지 버전을 작게 표시 (예: `FC Sky v0.1.11`).
- `APP_VERSION` 을 단일 소스로 주입:
  - `deploy/Dockerfile` — `ARG APP_VERSION`(릴리스 버전) → LABEL·런타임 `ENV APP_VERSION` 동시 반영.
  - `config/settings.py` — `APP_VERSION = os.environ.get('APP_VERSION', '')`.
  - `config/context_processors.py` — `app_version` 컨텍스트 추가(TEMPLATES에 등록).
  - `templates/base.html` — 값 있으면 `v{{ app_version }}` 표기.

## 비고
- 릴리스 시 버전 변경 위치: Dockerfile `ARG APP_VERSION` + compose `image:` 태그.

## 테스트
- 컨테이너 `APP_VERSION=0.1.11`, navbar에 `v0.1.11` 렌더, 홈 200.
