# 서초 K7 전체 일정(1~15번) 시드 + 홈/원정 교정

2026-07-13

## 내용

리그 공식 일정표(사용자 제공 스크린샷) 기준으로 `seed_seocho_k7` 를 전체 리그로 확장.

- **전체 15경기**(6팀 싱글 라운드로빈, 6/28·7/12·8/1·9/5·9/19 각 3경기)로 시드 데이터
  확장. 비고(note)에 공식 경기 번호("N번경기") 기록. 확보된 결과 반영:
  1번 ACI 3:7 오키나와, 2번 리얼 1:1 시누쓰, 3번 스카이 1:1 HUMBLE,
  4번 시누쓰 0:2 스카이. 5·6번(7/12)은 종료·스코어 미확보.
- **홈/원정 교정**: 4번경기는 스카이가 원정(초기 시드는 스카이 홈으로 잘못 입력,
  시간도 11:30→11:20). 시드가 같은 날짜의 **뒤집힌 대진을 감지하면 참가팀·스코어를
  함께 스왑**해 교정한다 — 운영 DB 재실행 시 중복 생성 없이 바로잡힘.
- 기존 경기엔 빈 비고 채움. `docs/seocho-k7-schedule.md` 를 전체 일정표(번호·결과
  컬럼 포함)로 재작성.

## 검증

- 운영 미러(dev_data, 어제 운영 시드 반영본) 사본에 재실행: 신규 9·갱신 6
  (4번경기 "홈/원정 교정·일시·비고 갱신" 확인), 재실행 멱등(0·0).
- 4번경기 최종: 시누쓰(홈) 0:2 스카이(원정), 스카이 관점 result=W.
- 홈·대회 상세 렌더: None 없음, 최근 결과에 "시누쓰 vs **스카이** 0 : 2 승".

## 운영 반영

이미지 0.6.10 배포 후: `docker exec <컨테이너> python manage.py seed_seocho_k7`
(5·6번경기 결과는 확인되는 대로 웹 '편집'으로 입력).
