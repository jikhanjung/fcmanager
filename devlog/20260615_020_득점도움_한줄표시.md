# 20260615_020 — 득점-도움 한 줄 표시 (경기 타임라인)

## 변경
- 경기 상세 타임라인에서 **득점과 그 도움을 한 줄**로 묶어 표시.
  예: `33' 득점 김민성 (도움: 박주성)`
- `apps/matches/views.py` — `_build_timeline()` 추가. GOAL과 ASSIST를
  **같은 팀(side)·같은 분(minute)** 기준으로 페어링, 짝 없는 도움/그 외 이벤트는 단독.
- `templates/matches/match_detail.html` — `events` → `timeline`(주 이벤트 + 도움) 렌더.

## 비고
- GOAL/ASSIST는 별도 MatchEvent라 명시적 링크가 없어 (side, minute)로 페어링.
  분을 입력하면 정확히 매칭되고, 분 미입력(과거 시드)은 순서 기준 best-effort.
- 득점/도움 집계(순위·통계)는 이벤트 단위라 영향 없음 — 표시만 변경.

## 테스트
- match 6(골3·도움3) → 타임라인 3줄로 묶임, 페이지 200, 도움 인라인 표시 확인.
