# 서초 K7 6/28 경기 이벤트·출전명단 시드 (KFA 기록지)

2026-07-13

## 내용

KFA 통합경기정보 기록지 PDF 3건(1~3번경기)에서 득점·도움·경고·교체·명단을 추출해
`seed_seocho_k7` 에 추가. 경기 매칭은 비고(note)의 경기 번호, **이미 이벤트/명단이
있는 경기는 건너뜀**(콘솔·수동 기록 보존 + 멱등).

- **이벤트 52건**: 1번(ACI 3:7 오키나와) 골 10·도움 1·교체 12, 2번(리얼 1:1 시누쓰)
  골 2·교체 14, 3번(스카이 1:1 HUMBLE) 골 2·경고 3·교체 8.
- **선수 연결**: 우리 팀(스카이)은 이름으로 Player 매칭(`NAME_ALIASES`: KFA 동명이인
  표기 "이상호1"→"이상호"), 외부팀 선수는 이름을 description 에 기록.
- **도움 연결**: 같은 분 골 → 없으면 직후 골에 링크(1번경기 Shin Jae Ho 도움 32'는
  39' 성찬호 골에 연결 — 기록지 분은 32' 그대로 저장).
- **상대팀 간 경기 side 컨벤션**: 홈팀 → OUR(타임라인 왼쪽), 원정팀 → OPPONENT(오른쪽).
  side 는 정렬용일 뿐, 통계(scorers/stats)는 player 지정 이벤트만 집계하므로 오염 없음.
- **출전 명단(3번경기 스카이)**: 선발 11 + 벤치 23, 주장 박찬영. DB 명단에 없는
  선수(신광렬)는 제외하고 로그로 알림.
- 템플릿(`match_detail.html` 타임라인): 선수 미연결 시 description(외부팀 선수 이름)을
  이름 자리에 표시, 상대팀 간 경기에선 "우리 팀/상대팀" 라벨 대신 참가팀 이름 표시.

## 검증 (운영 미러 사본)

이벤트 52건·명단 34명 기록, 재실행 시 전부 건너뜀(멱등). 1번경기 상세: 외부팀 득점자
이름 표시·도움 39' 골 연결·'우리 팀' 라벨 없음. 3번경기: 박주성 골 Player 연결·경고 3·
교체 전원 매칭·선발 11+주장. 득점 순위에 박주성 반영, 외부팀 선수 미포함. 테스트 50건 통과.

## 미확정 (추후 입력)

- HUMBLE 측 경고(기록지 분 표기 잘림) — 미기록.
- 5번(HUMBLE-ACI)·6번(오키나와-리얼) 스코어, 4번(시누쓰-스카이) 상세 기록지 — 미확보.
