## 한 일

구청장기 구조(조별리그 → 녹아웃)를 모델로 담고, 대회 상세에 대진표를 추가했다.

### 표 헤더 가독성 (빠른 수정)
- 대회 관리·선수 관리 표의 `<thead>` → `table-light`(어두운 배경+어두운 글씨로 안 보이던 문제).

### 경기 단계 stage
- `Match.stage` 추가(마이그레이션 0009): 조별리그(GROUP)/16강/8강/준결승(SF)/3·4위전/결승(F).
  기본값 GROUP. `STAGE_ORDER`·`KNOCKOUT_STAGES`·`is_knockout`·`stage_order` 헬퍼.
- 데이터 백필(0010): 기존 경기의 note 라운드 표기로 stage 설정. 결승 note 설명에 '준결승 승자'가
  들어 있어 **'—' 앞부분(라운드)만 판정**하도록 처리 → 준결승 2·결승 2 정확히 분류.
- `load_seocho_knockout`: round→stage 매핑으로 stage 지정.

### 표시
- **순위표**: 조별리그(GROUP)만 집계(녹아웃 제외) → 조별 순위 정확.
- **대회 상세**: 경기를 '조별리그 경기' + '녹아웃 대진표'로 분리. 대진표는 부문별 → 단계별
  열(준결승→결승)로 카드 배치(우리 경기 기준 진행). CSS 가로 스크롤 bracket.
- 경기 목록·상세에 녹아웃 단계 배지.
- 경기 편집 폼(MatchResultForm)·Match admin에 stage 추가.
