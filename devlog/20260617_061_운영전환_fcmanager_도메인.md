# 20260617_061 — 운영 전환 준비: fcmanager.app 전용 도메인 (SaaS)

> 멀티테넌트(Phase 6) 운영 반영. 도메인 `fcmanager.app` 확정.

## 결정
- SaaS 앱을 **전용 도메인 루트**에서 서비스: `https://fcmanager.app/`(플랫폼) + `/<slug>/`(클럽).
- 기존 `/FcSky/` 서브패스(공유 호스트) 방식 폐기 → `DJANGO_URL_PREFIX` 미사용.

## 코드/이미지 준비 (이번 커밋)
- **compose 2종**: `DJANGO_URL_PREFIX` 제거(정적/미디어 → `/static/`,`/media/`),
  `ALLOWED_HOSTS`/`CSRF_TRUSTED_ORIGINS` 에 `fcmanager.app`·`www.fcmanager.app`(https) 반영.
  이미지 태그 `0.5.7 → 0.6.0`.
- **Dockerfile**: `APP_VERSION=0.6.0`, URL_PREFIX 주석을 SaaS(루트) 기준으로(ARG 기본 빈 값 유지).
- **deploy.sh**: 헬스체크 경로 `/FcSky/admin/login/ → /admin/login/`.
- **nginx 템플릿**: `deploy/host/nginx-fcmanager.conf.example`(proxy `/`·`/media/` alias·certbot 안내·`/FcSky/`→`/fcsky/` 리다이렉트 메모).
- **deploy.md**: "SaaS 도메인 전환(1회)" 절차 추가.

## 운영 적용 순서 (dolfinid에서, devlog/deploy.md 참고)
1. DNS: fcmanager.app·www → 34.64.158.160 A 레코드.
2. (m710q) 0.6.0 이미지 build/push (URL_PREFIX 없이 → 정적 /static/).
3. nginx fcmanager 블록 + `certbot --nginx`.
4. `sync_to_srv` → `/srv/FcSky/.env` 에서 `DJANGO_URL_PREFIX` 제거 → `deploy.sh 0.6.0`.
5. 검증: `https://fcmanager.app/`·`/fcsky/`.

## 주의
- 컨테이너 교체와 nginx 변경은 **한 묶음**. 이미지만 갈면 기존 `/FcSky/` 깨짐.
- 이번 커밋은 준비만(빌드/푸시 별도). 운영 nginx·DNS·인증서는 dolfinid에서 수행.
