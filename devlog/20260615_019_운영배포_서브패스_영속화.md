※ nginx 설정·`/srv/FcSky`·`deploy/.env` 는 git 비추적(서버 로컬).

## nginx (서버 로컬, /etc/nginx/sites-available/2026biennale)
- 포트 80 default_server 블록에 `location /FcSky/` → `proxy_pass http://127.0.0.1:8003`
  (Django가 `DJANGO_URL_PREFIX=FcSky`로 접두사 자체 처리 → 접두사 **보존**, 스트립 안 함).
- `location /FcSky/media/` → `alias /srv/FcSky/media/` (선수 사진 등 직접 서빙).

## compose (deploy/docker-compose.yml)
- host **8003**→컨테이너 8000 (호스트 8000은 portainer 점유).
- **DB·미디어 영속화**: `/srv/FcSky/db.sqlite3`, `/srv/FcSky/media` bind-mount
  (uid 1000=컨테이너 appuser 소유). 기존 named volume 제거.
- `DJANGO_SEED=0`(기본) — 영속 DB 데이터 보존, 초기 적재 때만 1.
- `DJANGO_ALLOWED_HOSTS`/`CSRF_TRUSTED_ORIGINS`에 IP 추가(DEBUG=false IP 접근).
- **운영 SECRET_KEY**는 git 비추적 `deploy/.env` 에서 `env_file`로 주입(레포에 비밀 미포함).
