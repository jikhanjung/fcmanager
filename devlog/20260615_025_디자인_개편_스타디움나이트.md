# 20260615_025 — 디자인 전체 개편 (Stadium Night 테마)

frontend-design 스킬 기반으로 사이트 전체 룩앤필을 세련되게 개편.

## 컨셉
- **야간 경기장(Stadium Night)**: 딥 네이비 + 일렉트릭 스카이블루 + 골드 포인트.
- 폰트: **Pretendard**(한글 본문) + **Bebas Neue**(스코어/숫자/워드마크, 스코어보드 느낌).

## 변경
- **`static/css/fcsky.css`** 신규 — CSS 변수 토큰, Bootstrap 색 토큰 오버라이드,
  네비/카드/버튼/배지/테이블/폼/리스트/알림/푸터 전역 스타일, 히어로,
  등장 애니메이션(staggered reveal, prefers-reduced-motion 대응).
- **base.html** — Pretendard·Bebas·fcsky.css 로드(+`extra_head` 블록).
  네비 `site-nav`(스티키·글래스·언더라인 호버) + `FC SKY` 워드마크 + 버전 첨자.
  푸터를 네이비 풀폭 디자인으로.
- **home.html** — 드라마틱 **히어로**(플러드라이트 그라디언트 + 잔디 라인 패턴,
  대형 Bebas 타이틀, CTA, reveal 애니메이션) + 섹션 타이틀(액센트 바) + 스코어 숫자.
- 전역 CSS라 팀·선수·경기·순위·공지 등 **모든 페이지가 새 카드/버튼/배지/테이블 스타일**을 자동 반영.

## 제약 준수
- Bootstrap 5 CDN 유지, 별도 JS 빌드 없음(폰트는 CDN, CSS는 1개 정적 파일).
- 정적파일은 WhiteNoise manifest 해시 + nginx 미디어와 별개로 정상 서빙.

## 테스트
- `fcsky.<hash>.css` 200(text/css) 서빙, 홈/팀/경기/순위/공지 200,
  히어로·워드마크·스코어 마크업 렌더 확인.

## 배포
- `honestjung/fcsky:0.2.2`.
