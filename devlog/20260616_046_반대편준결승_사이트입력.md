# 20260616_046 — 반대편 준결승 결과 사이트 입력 화면

## 한 일

[[20260616_045_대진자동진행_feeder]]의 후속. 결승 상대를 자동 채우는 **반대편 준결승
(상대팀 간 경기)** 결과를 admin이 아닌 **사이트에서 입력**하는 화면을 추가했다.
(우리 준결승은 기존 경기 편집/라이브 콘솔로 입력 가능 — 이번엔 OpponentMatch 입력 보강.)

- `OpponentMatchResultForm`(matches/forms): home/away 점수 + 일시·비고.
- `opponent_match_edit` 뷰(staff) — `matches/opponent/<pk>/edit/`. 저장 시 post_save
  시그널이 연결된 결승 상대를 자동 기입. `?next=`로 원래 페이지 복귀.
- `templates/matches/opponent_match_edit.html` — 양 팀 점수 입력 화면.
- 대회 상세 대진표: 준결승 열에 **반대편 준결승 카드** 표시(승자 굵게). staff에게만
  **'결과 입력'** 링크(→ opponent_edit, next=현재 대회 상세).

## 검증 (DB 사본)

- `check` 무이슈, 모델 변경 없음(마이그레이션 불필요).
- 대진표에 '반대편 준결승' 표기는 누구나, **'결과 입력' 링크는 staff만**(익명 미노출).
- staff가 입력(매현 2:0 신반포) → 302 복귀, **결승(40대) 상대 자동 '매현'**.
- 익명 edit 접근 → 302(로그인).
- 0.5.4 배포 후 라이브 동일 확인, navbar v0.5.4.

## 운영 사용법

- 대회 상세(모든 대회 → 구청장기) 대진표에서 반대편 준결승의 **'결과 입력'** 클릭 →
  점수 저장 → 결승 상대 자동. 우리 준결승은 경기 편집/라이브 콘솔로.
