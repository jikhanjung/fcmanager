# 20260616_048 — 녹아웃 대진표 연결선 버그 수정

## 문제

[[20260616_047_대진표_비주얼_부문탭]]의 대진표 연결선이 이상하게 보임.

원인: 준결승 칸(`.ko-col--semis`)의 **첫 자식이 칸 제목(`.ko-col-title`)** 이라
`.ko-cell:first-child`가 첫 준결승 카드에 적용되지 않음 → 첫 준결승의 가로선·세로선
일부가 누락/오배치됐다.

## 한 일

- 준결승 두 카드를 `.ko-pair`로 감싸 `:first-child` 의존 제거.
- 연결선 재작성:
  - 각 준결승 카드 → 오른쪽 가로선(`.ko-pair .ko-cell::after`).
  - 두 가로선 끝을 잇는 **하나의 세로선**(`.ko-pair::after`, 25%~75% = 두 카드 중앙).
  - 세로선 중앙에서 결승으로 가로선(`.ko-col--final .ko-cell::before`).
- 두 토막 세로선을 잇던 방식 → 단일 세로선으로 깔끔하게(이음매 제거).
- 카드 테두리 가독성: 옅은 회색(`--bs-border-color`) → 1.5px `#8c98a6`로 또렷하게.
  결승 카드 강조도 진한 골드(`#e0a800`).

## 검증

- `check` 무이슈, 모델 변경 없음.
- 구청장기 상세 200, `.ko-pair` 안에 준결승 2칸 + `.ko-col--final` 결승 1칸 구조 확인.
- 0.5.6 배포 후 라이브 200, v0.5.6.
