# 20260615_009 — 상대팀 간 경기 모델(조 순위 보정)

## 배경
순위표(조별)는 우리 팀 경기만으로 집계 → 상대팀 간 경기(예: 신반포 vs 방현)가 빠져
조 전체 순위가 불완전했음. 사용자가 50대 A조 신반포 4:2 방현 결과 제공.

## 변경
- **모델 `OpponentMatch`** (matches): competition·season·age_group(부문)·home/away(Opponent)·
  home_score/away_score·kickoff·note. 우리 팀이 끼지 않은 경기 저장용. + admin 등록, 마이그레이션.
- **standings 뷰**: 우리 팀 경기로 부문별 조 표를 만든 뒤, 같은 `age_group`의 `OpponentMatch`를
  해당 조에 합산(home/away 양측 전적). 안내 문구도 "입력 시 반영"으로 갱신.
- **seed**: `OPPONENT_MATCHES`에 신반포 4:2 방현(50대) 추가.

## 검증
- 마이그레이션·시드 OK(상대간 1).
- 50대 A조 완성: 신반포 6점(2승)·스카이 3점(1승1패)·방현 0점(2패). 신반포 4:2 방현 반영 확인.
