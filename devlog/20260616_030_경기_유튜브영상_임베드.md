# 20260616_030 — 경기 페이지 유튜브 영상 임베드 (여러 개)

## 추가
- **`MatchVideo`** 모델(`match` FK, `url`, `title`) — 경기당 여러 영상. 마이그레이션 `matches.0004`.
  - `extract_youtube_id()` — watch/youtu.be/shorts/embed/live + 순수 ID 형식 지원.
  - `youtube_id`/`embed_url` 프로퍼티.
- 경기 상세(`match_detail.html`): 영상이 있으면 **반응형 16:9 임베드**(Bootstrap `ratio`)로 표시,
  여러 개는 2열 그리드. `loading="lazy"`, allowfullscreen.
- 운영진 편집(`match_edit.html`): "경기 영상" 폼셋(url+제목) 추가, **"+ 영상 추가"** JS(행 복제 일반화),
  잘못된 유튜브 링크는 폼 검증으로 거부.
- `views.py` — `match_edit`에 `MatchVideoFormSet` 처리, `match_detail`에 `videos` 컨텍스트.
- admin: `MatchVideo` 등록 + Match 인라인.

## 테스트
- 링크 형식별 embed_url 추출/검증, 폼셋 저장, 상세 페이지 iframe 2개 렌더, 편집 GET 영상 카드,
  통합 POST(이벤트+영상 동시 저장) 302 확인. 테스트 영상은 정리(match 5 영상 0).
