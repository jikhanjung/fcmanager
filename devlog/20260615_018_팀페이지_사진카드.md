# 20260615_018 — 팀 페이지 선수 사진 카드

## 변경 (templates/teams/team_detail.html)
- 선수단을 표 → **사진 카드 그리드**(KFA 참가신청 페이지 스타일)로 변경.
- 카드: 원형 사진(없으면 이름 이니셜 아바타) + 이름 + `No.등번호` + 주장 C 배지,
  포지션·출생연도·(있으면)구분(50대초/말).
- 반응형(모바일 1 · 태블릿 2 · 데스크톱 3열), hover 시 살짝 떠오르는 효과, 카드 전체 링크.

## 비고
- 공유 템플릿이라 사진 없는 선수(타 팀)는 이니셜 아바타로 안전하게 대체.

## 테스트
- sky-k7 200, 선수 카드 35 + 사진 35 렌더, 인원수 표기, 기존 표 제거 확인.
