# 20260615_014 — 탑메뉴 드롭다운 재구성

## 배경
- 공지·갤러리·아카이브·로그인 등이 추가되며 네비 항목이 10개로 늘어 평면 나열이 번잡.

## 변경 (templates/base.html)
- 탑메뉴를 **홈 · 공지 · 팀 · 대회 · 아카이브 · (로그인/관리)** 6개로 축소.
  - **대회 ▾**: 일정·결과 / 순위표 / 득점 순위 / 통계
  - **아카이브 ▾**: 시즌 아카이브 / 명예의 전당 / 갤러리
- Bootstrap 5 드롭다운(`dropdown-toggle`/`dropdown-menu`), 번들 JS는 기존 로드분 사용.
- 운영진(is_staff) "관리" 링크를 하드코딩 `/admin/` → `{% url 'admin:index' %}` 로 수정
  (서브패스 `/FcSky/` 배포에서 깨지지 않도록).
- 저장 결과 알림용 **플래시 메시지 배너**(`messages`) 추가 — 경기 결과 편집 등에서 사용.

## 테스트
- 드롭다운 7개 항목 실제 href 전부 200, 익명/운영진 분기 정상.
