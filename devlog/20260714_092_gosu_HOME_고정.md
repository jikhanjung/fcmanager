# gosu 드롭 시 HOME=/tmp 고정 (0.6.21)

- 날짜: 2026-07-14
- 출처: 타 인스턴스 작업(3-repo 수렴 — fsis 0.5.82 에서 발원, cdGTS 동형) 검토 후 반영.

## 변경

- `deploy/docker-entrypoint.sh`: `exec gosu <uid:gid> env HOME=/tmp MPLCONFIGDIR=/tmp/mpl "$0" "$@"`
- gosu 는 su 와 달리 환경을 재설정하지 않아, root 시작 컨테이너가 드롭한 뒤에도
  `HOME=/root`(비쓰기)가 남는다. HOME 에 쓰는 라이브러리(matplotlib 캐시 등)가 나중에
  추가되면 잠복 크래시 — 선제 차단. `env -i` 아닌 `env` 라 기존 환경(DATABASE_PATH 등) 보존.

## 검토 확인

- cdGTS `entrypoint.sh` 와 자구 동일, fsis2026 은 `DROP` 변수로 같은 패턴 — 3-repo 수렴 성립.
- fcmanager 는 Pillow 뿐(matplotlib 없음)·앱 코드 HOME 미사용 → 현시점 동작 무변화. `sh -n` 통과.
