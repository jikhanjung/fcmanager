# 20260622_069 중계 연장전·승부차기 + 시청자 진행시계

## 배경
전후반만 있던 중계 콘솔에 녹아웃 경기의 **연장전·승부차기**가 필요. 함께 시청자
페이지에 진행 시계·갱신 카운트다운·LIVE 깜박임을 추가.

## 결정 사항 (사용자 확정)
- **연장**: 전·후반 기본, 시계는 정규 풀타임부터 **이어서**. **단일 연장**도 가능(대회/부문 override).
- **연장 길이**: `extra_half_minutes`(대회 기본 15 + 부문 오버라이드), 대회 편집 폼 노출.
- **승부차기**: 킥 하나하나(성공/실패) 기록 → 성공 수 합산, 동점 시 승자 판정.
- **노출**: 녹아웃(is_knockout) 경기만. **연장 없이 바로 승부차기**도 가능.

## 변경
### 모델 (matches 0017, competitions 0010)
- `Match.Period` 확장: `ET_FIRST/ET_HALFTIME/ET_SECOND/PENALTIES`.
- `Match`: `et_first_started_at`·`et_second_started_at`·`home_pso_score`·`away_pso_score`.
- `Match.winner_entry` 가 본점수 동점 시 승부차기로 판정, `decided_by_penalties`·
  `our/opponent_pso_score`·`extra_half_*`·`extra_time_single`·`extra_time_total_seconds` 프로퍼티.
- `MatchEvent.EventType`: `PSO_GOAL`/`PSO_MISS`. `half` 는 3=연장전반·4=연장후반.
- `Competition`/`Division`: `extra_half_minutes`·`extra_time_single`(부문은 null=대회값).

### 시계 (`_elapsed_seconds`)
정규(2×half)부터 이어서: 연장 전반 = reg + 경과, 연장 휴식 = reg+ehalf 정지,
연장 후반 = reg+ehalf + 경과, 승부차기/종료는 nominal. 일시정지(paused_seconds)도 연장 적용.

### 콘솔 (`match_live.html`)
- 단계 흐름에 연장/승부차기. 후반·연장후반 종료 시 **녹아웃+동점이면 "▾ 다음 단계" 시트**
  (연장 시작 / 승부차기 / 경기 종료) — 연장 건너뛰고 바로 승부차기 선택 가능.
- 승부차기 단계엔 **PSO 패널**(우리/상대 성공·실패 4버튼) + 실시간 승부차기 스코어,
  일반 동작 자동 숨김. 연장/PSO 단계 배지.

### 점수/집계 (`services.recompute_score`)
승부차기 킥 이벤트가 있으면 성공 수로 `home/away_pso_score` 집계(없으면 None).

### 시청자 (`match_detail.html`)
- **진행 시계**(mm:ss, 콘솔과 동일 보간·연장 이어가기), **단계 라벨**, **갱신 카운트다운**
  (`🔄 N초 후 갱신`), **LIVE 배지 깜박임**(CSS `liveBlink`).
- 승부차기 점수·"승부차기 승/패" 배지(동점 PSO 결정 시).

### 폼/Admin
- 대회 편집 폼에 `연장 길이(분)`·`연장 단일 진행` 체크박스. Admin 에 연장 필드·PSO 점수 노출.

  승부차기 집계/승자 판정·PSO 없으면 None).
