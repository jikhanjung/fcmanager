# 20260615_026 — 경기 편집: 득점 행에서 도움 동시 입력

## 목표
- 득점자를 입력할 때 **같은 줄에서 도움 선수도 함께** 선택.

## 변경
- `apps/matches/forms.py` — `MatchEventForm`에 모델 외 필드 `assist_player`(해당 팀 선수로 제한).
  편집 시 득점 행에 짝지어진 기존 도움(같은 side·minute)을 초기값으로 표시.
- `apps/matches/views.py` — `match_edit`:
  - 이벤트 폼셋 queryset에서 **ASSIST 제외**(도움은 별도 행으로 안 보임).
  - 저장 시 기존 ASSIST 전부 삭제 후, 각 득점(GOAL) 행의 `assist_player`로 ASSIST 이벤트
    재생성(같은 side·minute) — 멱등, 고아/중복 없음.
- `templates/matches/match_edit.html` — "도움(득점 시)" 열 추가(기존 행·빈 행·JS 추가행).

## 비고
- 도움은 득점 행에서만 관리(별도 입력 행 없음). 타임라인 표시는 기존 (side, minute) 페어링
  그대로라 "득점 OOO (도움: XXX)" 한 줄로 나옴.
- 득점/도움 순위·통계는 GOAL/ASSIST 이벤트 단위 집계라 영향 없음.

## 테스트
- 득점+도움 한 줄 POST → GOAL·ASSIST 두 이벤트 생성(같은 분/팀), 재편집 시 도움 초기 선택
  유지, 도움 별도 행 미표시 확인.

## 배포
