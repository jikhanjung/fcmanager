# 20260615_007 — 토너먼트 조별(부문별) 순위표

## 배경
기존 순위표는 LEAGUE 대회만 대상 + 우리 팀 행만 표시 → 리그 경기 데이터가 없어
화면이 비어 있었음. 토너먼트(서초구청장기)는 우리 경기 데이터가 있으나 표시 안 됨.

## 변경
`competitions.standings` 뷰를 일반화:
- 대상: **경기 기록이 있는 모든 대회**(리그·토너먼트). 기본값은 종료 경기가 있는 대회.
- **부문(= 우리 팀)별로 조 순위표**를 생성. 각 표는 우리 팀 + 그 상대팀들을 포함하며,
  **우리 팀 경기 기준**으로 양측 전적을 집계(우리 승 → 상대 패 등).
- 승점 = 승*3 + 무*1. 정렬: 승점→득실차→득점→이름.
- 부문 정렬: K7 → 40대 → 50대. 우리 팀 행은 강조(table-primary).

## 한계 (명시)
상대팀 간 경기(예: 신반포 vs 방현)는 데이터가 없어 미반영 → 조 전체 순위와 다를 수
있음. 화면 하단에 안내.
