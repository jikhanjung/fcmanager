# 20260615_024 — 경기 편집: 득점/도움 선수 목록을 팀 소속으로 제한

- 경기 결과 편집 폼의 이벤트 줄에서 **선수 선택지를 해당 경기 팀(our_team)에 등록된

- `apps/matches/forms.py` — `MatchEventForm.__init__(team_players=...)`: 주어지면
  `player` 필드 queryset 교체.
- `apps/matches/views.py` — `match_edit`에서
  `Player.objects.filter(memberships__team=match.our_team).distinct()` 를
  `MatchEventFormSet(..., form_kwargs={"team_players": qs})` 로 전달.
  기존 줄·빈 줄(JS 추가행, empty_form)에 모두 적용됨.
